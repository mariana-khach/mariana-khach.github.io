About me

Data Scientist with Physics PhD

Welcome! I’m Dr. Mariana Khachatryan, a Data Scientist based in the Hampton Roads are in Virginia, originally from Yerevan, Armenia. My passion for physics, programming, and data science drives everything I do blending analytical rigor with creativity to extract insights and build intelligent solutions.
Growing up in a family of scientists, I naturally gravitated toward discovery and problem-solving. During my bachelor’s studies at Yerevan State University, I attended the 2010 DESY Summer School in Germany, where I completed my first data analysis project and presented results to both technical and non-technical audiences. This experience sparked my lifelong interest in using data to tell meaningful stories and drive decisions.
My journey continued through graduate research at Jefferson Lab and Old Dominion University, where I earned my Ph.D. in Physics in 2019 and later pursued a postdoctoral fellowship at Florida International University. Throughout my academic career, I worked with large-scale datasets, developing and implementing data science and analytics solutions to extract actionable insights, led analytics projects, mentored junior researchers, earned multiple awards and recognition from Jefferson Lab and ODU, produced results that were published in Nature featuring in multiple media outlets (JLab, ODU1, ODU2).
Transitioning to industry, I’m excited to apply my research background and quantitative expertise to solving real-world challenges. As part of the Erdős Institute’s Data Science Bootcamp (Fall 2024), I built machine learning models, defined Key Performance Indicators (KPIs), and communicated findings to product stakeholders. I also hold an Udemy Nanodegree in Deep Learning, Artificial Intelligence, and Generative AI, reflecting my continued commitment to learning.
Outside of data and research, I find balance through traveling, gardening, and gaming on Nintendo Switch, activities that help me stay curious, creative, and energized.

Data Science & Analytics Projects

Here you can see some of the projects that I have worked on throughout my career.

Car price prediction

Led a team of PhD researchers to develop supervised learning models (Linear Regression, SVM, XGBoost) for predicting car prices using structured automotive data. Used Exploratory Data Analysis and Feature Engineering to improve model performance, achieving R² of 0.88. Applied SHAP values to identify key features driving model prediction. Delivered technical findings and business implications to product stakeholders from partner companies (Carmax, Upstart, etc.).

Clothing Sales Forecasting with LSTM and Linear Regression

Developed time series forecasting models using LSTM networks and Linear Regression to predict clothing sales trends. Engineered features to describe time dependent and serially dependent properties of Time Series to enhance predictive power of Linear Regression model. Achieved R² score of 0.91, demonstrating high accuracy on real-world retail data. Compared model performances to establish tradeoffs between interpretability and predictive strength.

Developed PDF chatbot using OpenAI API.

Have used OpenAI API and LangChain framework for developing applications powered by large language models (LLMs), to develope a PDF chatbot that uses natural language processing (NLP) to understand queries and OpenAI API to extract email address from provided PDF resume.

LLM Fine tuning and serving with FastAPI

Fine-tuned model for text classsification using the OpenAI API and served with FastAPI.

Facemask identification

Used convolution neural network (CNN) from Keras/Tensorflow to classify images with/without a face mask to 97% accuracy.

Tweet classification

Performed text processing and classification using NLP techniques, leveraging TensorFlow LSTMs and a Naive Bayes model.

Extraction of a signal corresponding to pi1 exotic meson via multivariate analysis of GlueX experiment particle physics data at Jefferon Lab

Used Python and C++ for exploratory data analysis, data engineering and predictive analysis of particle physics data at Jefferson Lab. Developed data analysis strategy and timeline. Used Least Squares and unbinned Likelihood statistical methods to clean high-volume data via Probabilistic Event Weightings. C++ to classify particles and identify exotic mesons via Unbinned Maximum Likelihood fitting. Used bootstrapping for error estimation. Delivered presentations to technical and non-technical audiences. Mentored team members.

Image generation for handwritten digits.

Generated images of handwritten digits using Variational Autoencoders (VAEs) and Generative Adversarial Networks (GANs).

Image color quantization via K-Means clustering.

Have used K-Means clustering unsupervised learning algorithm to reduce number of colors in sailboat image to three.

Comparison between K-Means and Density-Based Spatial Clustering (DBSCAN) on moon shape clusters.

Comparison between K-Means and DBSCAN on moon shape clusters. I have apply DBSCAN on clusters with irregular shape and have shown that it gives better performance than K-Means clustering.
